# octra-devnet-cli-setup
A complete well-detailed walkthrough for installing and using the Octra terminal client (from octra-labs/octra_pre_client) on the Devnet network.
Octra is an FHE-powered blockchain for encrypted computations. This guide covers:
- Prerequisites (Python 3.8+, Git)
- Cloning & virtual environment setup
- Installing dependencies
- Configuring your wallet.json (private key + address)
- Running the TUI client
- Basic commands: check balance, send OCT, multi-send, view tx history
- FHE basics: encrypt/decrypt balances (for testnet tasks)
- Troubleshooting common issues (venv activation, Windows/Linux differences)
- Extra tips: faucet claims (faucet-devnet.octra.com), explorer (devnet.octrascan.io), and testnet participation
